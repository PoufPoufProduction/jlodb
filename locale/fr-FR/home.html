Présentation

jLoDB est une base de données regroupant un ensemble d'activités éducatives libres. Ces activités sont référencées au sein de la base suivant leur niveau, leur difficulté ainsi que leur champ d'application, entre autres. Elles sont également structurées à l'identique afin de faciliter leur intégration au sein de logiciels tiers.

La technologie utilisée repose sur le langage Javascript. Celui-ci offre un nombre considérable d'avantages : format ouvert, technologie mature et reconnue, format SVG nativement intégré. En outre la nature scripturale du langage permet d'encapsuler du code dans les paramètres des exercices offrant des possibilités sans limites.

La base de données dans son intégralité est sous licence GPL-3.

Le saviez-vous ?

La base de données ainsi que l'ensemble des activités qui y sont référencées s'adaptent à n'importe quelle résolution d'écran. Seul le format 4/3 paysage est imposé.

Quel que soit la nature de l'activité, l'utilisation d'un clavier reste facultative. jLoDB est ainsi compatible avec une utilisation sur tablette ou TBI.

Afin de résoudre tout problème dû à la résolution, mais aussi, dans le but d'offrir d'avantage de liberté d'utilisation, la totalité des ressources graphiques est définie grâce à un format vectoriel.

Technologies utilisées

Dans un souci d'accessibilité, jLoDB est basée sur les technologies web, à savoir : HTML5, CSS3, javascript.

Chaque activité, du moment qu'elle se conforme à un certain nombre de règles, peut être développée avec n'importe quelle bibliothèque (ou pas, d'ailleurs). Aussi, l'utilisation actuelle de jquery et de ses dérivées (jquery-ui et jquery-svg) n'est pas une obligation en soit. Ce choix a été fait principalement pour uniformiser la structure technique des différentes activités.

MathML est utilisée pour l'affichage des formules mathématiques.
