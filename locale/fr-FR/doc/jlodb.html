jLoDb

Présentation générale

jLoDb (javascript Learning object Database) est un projet construit autour d'une base de données regroupant un ensemble d'exercices éducatifs libres. Ces exercices sont référencés au sein de la base suivant leur niveau, leur difficulté ainsi que leur champ d'application.

Cette base de données repose sur un ensemble de technologies web (HTML, CSS, Javascript, SVG, MathML) et elle est ainsi accessible depuis tout navigateur respectant ces protocoles. L'utilisation exclusive d'images vectorielles rend la solution indifférente à la résolution d'affichage. Seul le format 4/3 paysage est imposé. De plus, quel que soit la nature de l'activité, l'utilisation du clavier reste facultative.

jLoDB est compatible avec une utilisation sur ordinateurs, tablettes ou TBI. Enfin, la base de données dans son intégralité est sous licence GPL-3.

Les activités présentes dans l'environnement jLoDb sont des programmes permettant l'exécution d'exercices intéractifs. Chaque réponse proposée par l'utilisateur au cours de ces exercices est évaluée automatiquement. A la différence d'un cahier d'exercices qui peut nécessiter une vérification extérieure, l'utilisateur est ici totalement autonome.

JloDb est un programme libre pouvant être installé à peu près n'importe où. Une instance de démonstration est accessible à l'adresse suivante : http://jlodb.poufpoufproduction.fr/

Installation de JLoDb et de ses différents modules

jLoDb est un site web tout à fait classique nécessitant, pour fonctionner, d'un serveur web de type Apache, du langage PHP et du gestionnaire de bases de données MySQL. Ces trois logiciels libres sont généralement regroupés sous l'acronyme LAMP (avec 'L' pour Linux). Leur installation est largement documentée et elle est relativement aisée. Sous Ubuntu, la commande suivante permet l'installation complète du serveur. 

sudo apt-get install apache2 php5 mysql-server libapache2-mod-php5 php5-mysql

Une fois le serveur web correctement configuré, l'installation de jLoDb peut commencer.

Récupération du logiciel

Le code source de jLoDb est accessible depuis le compte GitHub de PoufPoufProduction à l'adresse suivante : https://github.com/PoufPoufProduction/jlodb

L'opération peut se faire en récupérant une archive du projet grâce au bouton Download ZIP ou en clonant directement le dépôt.

Attention, le répertoire res/img/svginventoryicons est un sous module dépendant du projet https://github.com/PoufPoufProduction/SVG-Inventory-Icons. L'opération clonage doit se faire avec l'option –recursive tandis que le chargement direct nécessite de charger également ce projet.

Une fois récupéré, le projet jLoDb doit être placé dans le répertoire de travail du serveur Apache (par défaut /var/www). Et la page localhost/jlodb/index.html doit renvoyer un premier résultat.

Configuration du logiciel

La configuration du site web se fait grâce au fichier conf/jlodb.ini

  • La section [admin] permet de définir un identifiant et mot de passe administrateur. Par défaut le couple login/password est admin/admin. Le mot de passe est crypté sous md5 mais de nombreux encodeurs sont disponibles sur le net : http://www.md5online.org/md5-encrypt.html par exemple.
  • La section [db] permet de définir les paramètres d'utilisation de mySQL. Le paramètre prefix permet d'ajouter un préfixe aux noms des tables définis par jLoDb, ce qui peut s'avérer utile dans le cas d'une base de données partagée.

Installation

Depuis un navigateur web compatible (Firefox est recommandé), ouvrir la page jlodb/index.html puis cliquer ensuite sur l'icône en forme de clé, en haut à droite de l'écran, pour s'identifier en tant qu'administrateur grâce au login/password définis dans le fichier de configuration. L'installation peut alors être lancée. Suivant les systèmes et la taille de la base, cela peut prendre quelques minutes.

Le site principal jLoDb est désormais installé. La page d'accueil doit désormais s'afficher correctement ainsi que le nombre d'activités et d'exercices disponibles dans la base.
