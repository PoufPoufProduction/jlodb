Présentation générale

jLoDb ou "Javascript Learning Object Database" est une base de données qui propose tout un ensemble d'activités éducatives libres. Ces activités sont référencées au sein de la base suivant de nombreux critères comme leur niveau, leur difficulté ou leur champ d'application. Un système de tags et de références croisées permet de conserver les synergies entre exercices. L'objectif est ici de pouvoir naviguer facilement parmi des exercices de plus en plus nombreux et d'y retrouver celui de son choix sans trop de difficulté.

Les exercices ou niveaux référencés dans la base sont structurés à l'identique afin de pouvoir être exécutés de façon indifférenciée, depuis la base elle-même ou depuis un logiciel tiers.

La base de données dans son intégralité est sous licence GPL-3. Cette intégralité comprend les différents programmes mais aussi les données des différents exercices ainsi que l'ensemble des ressources graphiques.


Installation de la base

jLoDb est une application web nécessitant, par conséquent, l'utilisation d'un serveur web de type AMP (Apache, MySql, PHP). Les possibilités sont nombreuses mais toutes sont désormais très classiques et sans réelle difficulté.

Serveurs web

AMP

Les logiciels Apache, MySql et Php forment un trio très connu dont l'installation a déjà été mainte fois détaillée. Il ne devrait y avoir aucun problème à procéder à leur installation.

Sous Windows, UvWamp est une version locale qui fonctionne sans installation particulière.

Sous Linux, l'installation peut se faire grâce à la commande suivante :

sudo install apache2 php libapache2-mod-php mysql-server php-mysql php-xml
